## Milestone Summary

**Decimal Phases:**
- Phase 4.1: Affiliate Reporting Bug Fixes (inserted after Phase 4 to close v1.1 audit gaps).

**Key Decisions:**
- Gate the admin queue-state label on `fundedDeferral` presence + status ∈ {pending,processing}, NOT `programAssigned` (unreliable — diagnostic payment shows `programAssigned=true` while queue is pending).
- `getQueueStateLabel` extracted to `_shared/paymentQueueLabel.ts` (not inlined) for clean 2-site sharing; admin components use hardcoded EN (no i18n) per repo convention.
