<objective>
Fix the dashboard Daily P&L Calendar widget so it reports correct daily profit per account/day instead of dropping orphan close deals.

Purpose: Trader on account 13535 sees +$55 for 2026-06-18 when the true closed P&L for that day is $20.16. Root cause: `mergedFromDeals` in `useTradingDashboardData.ts` does `continue` (silently skips) when a CLOSE deal has no matching OPEN deal in the loaded buffer (common at archive boundary / paginated loads). The orphan -$34.69 loss is dropped: $54.85 = $20.16 − (−$34.69). Ticket cmquy9bqo005pny0kw6j0lr71 (Trading Cult, URGENT).
