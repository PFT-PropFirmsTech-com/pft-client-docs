5. Query `messagetemplates.findOne({ event: "rule_breached" })`:
   - If `null` → console.log: `"[sync-rule-breached] No rule_breached template in DB ${dbName}. Skipping — templateAutoCreator will create on next backend boot."` + exit(0).

     a. Compute new variables array = union of `existing.variables` ∪ `NEW_VARIABLES` (preserve order: existing first, then any new ones not already present). Never remove.

6. Disconnect mongoose. Exit 0 on success.

7. Wrap in try/catch — on error, log and exit(1).
